# Chapter 20: Exercises & Mini-Projects

## 1. Spinning Lit Cube

**Goal**: Create a textured, lit cube that rotates

**Requirements**:
- 6 different colors or textures (one per face)
- Proper normals for lighting
- Smooth rotation
- Interactive camera

**Skills**: Immediate mode, normals, transformations

## 2. Textured Cube with Filtering

**Goal**: Compare texture filtering modes

**Requirements**:
- Load a texture
- Toggle between NEAREST, LINEAR, MIPMAP filters
- Display current filter mode
- Zoom in/out to see differences

**Skills**: Textures, mipmaps, text rendering

## 3. Teapot Material Showcase

**Goal**: Display teapot with different materials

**Requirements**:
- At least 5 material presets
- Multiple light sources
- Keyboard to switch materials
- Smooth lighting

**Skills**: Lighting, materials, GLU primitives

## 4. Particle System with Blending

**Goal**: Create a simple particle effect

**Requirements**:
- 100+ particles
- Alpha blending
- Simple physics (gravity)
- Render as GL_POINTS or textured quads

**Skills**: Blending, point sprites, arrays

## 5. Terrain from Heightmap

**Goal**: Render 3D terrain from heightmap

**Requirements**:
- Generate or load heightmap
- Use vertex arrays
- Texture mapping
- Lighting with calculated normals
- Camera controls

**Skills**: Vertex arrays, procedural geometry, normals

## Bonus Challenges

- **Solar System**: Hierarchical transformations (Sun → Earth → Moon)
- **FPS Counter**: Display real-time FPS
- **Fog Distance**: Implement distance-based fog
- **Skybox**: Create a textured skybox
- **Picking**: Implement mouse picking with ray casting
